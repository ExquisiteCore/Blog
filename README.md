# blog

**本项目是EC的个小站**

## 简介

一个简单的个人网站采用前后端分离架构

前端使用 Nuxt + Vue3 + TypeScript + shadcn-vue + Tailwind CSS 开发

后端使用 GoLang + GORM + Gin

数据库使用 postgresql
